# Javascript_advanced

This folder contains advanced JavaScript exercises and projects from the Holberton School front-end curriculum. The focus is on deepening our understanding of modern JavaScript concepts and techniques for dynamic web development.

## Contents

- **Advanced Functions:**  
  Practice with closures, callbacks, and higher-order functions.

- **Asynchronous JavaScript:**  
  Work with Promises, async/await, and handling asynchronous operations.

- **Object-Oriented Programming:**  
  Use ES6 classes, inheritance, and object manipulation.

- **DOM Manipulation:**  
  Select, modify, and create DOM elements using vanilla JavaScript.

- **Event Handling:**  
  Respond to user interactions such as clicks, form submissions, and keyboard events.

## Getting Started

1. Open the HTML files in your browser to see the exercises in action.
2. Review and modify the JavaScript files to practice and experiment.

## Requirements

- Basic knowledge of HTML and CSS.
- Familiarity with core JavaScript syntax and concepts.
